# Consejos y Trucos

Esta guía comparte técnicas avanzadas, atajos de teclado, flujos de trabajo eficientes y soluciones de problemas para ayudarle a dominar Gobify y trabajar más productivamente.

## Atajos de Teclado

Domine estos atajos para navegar Gobify más rápido.

### Atajos Universales

Estos funcionan en todo el sistema Gobify:

| Atajo | Acción | Contexto |
|----------|--------|---------|
| **Ctrl/Cmd + F** | Búsqueda rápida en página actual | Cualquier página |
| **Ctrl/Cmd + K** | Búsqueda global | Abre modal de búsqueda |
| **Esc** | Cerrar modal/diálogo | Modal activo |
| **Tab** | Moverse al siguiente campo | Formularios |
| **Shift + Tab** | Moverse al campo anterior | Formularios |
| **Enter** | Enviar formulario/confirmar acción | Formularios, diálogos |
| **Ctrl/Cmd + S** | Guardar formulario actual | Formularios de edición |
| **Ctrl/Cmd + R** | Actualizar página | Cualquier página |

### Atajos de Navegación

| Atajo | Acción | Va A |
|----------|--------|---------|
| **Alt + D** | Dashboard | Pantalla de inicio |
| **Alt + C** | Ciudadanos | Lista de ciudadanos |
| **Alt + P** | Programas Sociales | Lista de programas |
| **Alt + S** | Calles | Lista de calles |
| **Alt + N** | Colonias | Lista de colonias |
| **Alt + Q** | Escanear QR | Escáner QR |
| **Alt + R** | Reportes | Lista de reportes |

### Atajos de Tabla

Al ver tablas de datos:

| Atajo | Acción |
|----------|--------|
| **Teclas de flecha** | Navegar entre filas |
| **Enter** | Abrir fila seleccionada |
| **Espacio** | Seleccionar/deseleccionar fila |
| **Ctrl/Cmd + A** | Seleccionar todas las filas |
| **Page Up/Down** | Desplazar página |
| **Home** | Primera fila |
| **End** | Última fila |

**Nota**: Los atajos exactos pueden variar según el navegador. Los usuarios de Mac usan Cmd (⌘) en lugar de Ctrl.

## Flujos de Trabajo Eficientes

Optimice tareas comunes con estos flujos de trabajo optimizados.

### Rutina de Inicio Matutino

**Chequeo Diario de 5 Minutos (Recomendado):**

```
1. Iniciar sesión en Gobify
2. Revisar KPIs del Dashboard (30 segundos)
   - Notar cualquier cambio inusual
   - Identificar prioridades
3. Verificar reportes pendientes (1 minuto)
   - Filtrar: Asignado a Mí + Pendiente
   - Contar y priorizar
4. Revisar elementos urgentes (2 minutos)
   - Prioridad: Urgente
   - Estado: Todos excepto Cerrado
   - Abordar inmediatamente o delegar
5. Verificar inscripciones en programas esperando aprobación (1.5 minutos)
   - Sus programas asignados
   - Estado: Pendiente
   - Planear tiempo de procesamiento
```

### Procesamiento por Lotes de Ciudadanos

Cuando procese múltiples ciudadanos eficientemente:

**Fase de Configuración (2 minutos):**
```
1. Abrir sección Ciudadanos
2. Aplicar filtros relevantes
3. Personalizar columnas para mostrar solo lo que necesita
4. Establecer filas por página a 50
5. Ordenar por criterios deseados
```

**Fase de Procesamiento (por ciudadano: 1-3 minutos):**
```
1. Clic derecho en ciudadano → Abrir en nueva pestaña
2. Revisar detalles en nueva pestaña
3. Hacer actualizaciones necesarias
4. Guardar y cerrar pestaña
5. Regresar a lista principal (aún en misma posición)
6. Moverse al siguiente ciudadano
7. Repetir
```

**Por Qué Esto Funciona:**
- La lista principal permanece en su lugar
- No pierde su posición
- Navegación más rápida
- Navegación paralela posible

### Creación Rápida de Solicitud de Programa

**Optimizado para velocidad (5 minutos por solicitud):**

```
1. Tener documentos del ciudadano listos antes de comenzar
2. Usar segundo monitor o pantalla dividida:
   - Izquierda: Documentos físicos o escaneos
   - Derecha: Formulario de Gobify
3. Completar formulario de arriba hacia abajo sin detenerse
4. Usar Tab para moverse entre campos (no el mouse)
5. Búsqueda predictiva para calles/colonias
6. Mantener notas comunes como fragmentos de texto (copiar-pegar)
7. Cargar todos los documentos a la vez (selección múltiple)
8. Revisar en una pasada
9. Enviar
```

**Ahorradores de Tiempo:**
- Pre-escanear todos los documentos antes de comenzar formulario
- Verificar formato de CURP antes de escribir (evitar reingreso)
- Usar expansión de texto para frases comunes
- Mantener folios/referencias frecuentemente usados a mano

### Procesamiento de Múltiples Reportes

**Manejar alto volumen eficientemente:**

**Fase de Triaje (30 segundos por reporte):**
```
1. Filtrar: Estado = Pendiente
2. Ordenar: Prioridad descendente, luego Fecha ascendente
3. Revisar cada reporte:
   - Leer título
   - Notar tipo y categoría
   - Evaluar complejidad
4. Asignar rápidamente a personal (incluyéndose)
5. Establecer prioridad inicial
6. Moverse al siguiente
```

**Fase de Procesamiento (tiempo variable):**
```
1. Filtrar: Asignado a Mí + En Revisión
2. Comenzar con urgentes
3. Manejar elementos de resolución rápida primero (2-5 min cada uno)
4. Programar elementos complejos para tiempo enfocado
5. Actualizar todos los estados antes del descanso
6. Enviar notificaciones a ciudadanos por lotes
```

### Mantenimiento de Datos Geográficos

**Rutina de limpieza semanal (15 minutos):**

```
1. Sección Calles:
   a. Ordenar por "Número de Usos" ascendente
   b. Revisar calles con 0 usos
   c. Verificar duplicados obvios
   d. Eliminar/fusionar según sea necesario (2-3 minutos)

2. Sección Colonias:
   a. Proceso de revisión similar
   b. Verificar códigos postales contra lista oficial
   c. Corregir cualquier discrepancia (2-3 minutos)

3. Verificación de calidad:
   a. Buscar errores ortográficos comunes
   b. Verificar capitalización inconsistente
   c. Estandarizar nomenclatura (5 minutos)

4. Documentación:
   a. Anotar cambios realizados
   b. Actualizar al equipo sobre correcciones importantes (2 minutos)
```

## Técnicas de Usuario Avanzado

Métodos avanzados para usuarios experimentados.

### Flujo de Trabajo de Múltiples Pestañas

**Usar múltiples pestañas estratégicamente:**

**Pestaña 1: Dashboard**
- Siempre abierta
- Referencia rápida para KPIs
- Regresar aquí entre tareas

**Pestaña 2: Tarea Principal Actual**
- Ciudadanos, Programas, Reportes (en lo que esté enfocado)
- Mantener filtro/búsqueda activo
- Su pestaña de trabajo

**Pestaña 3: Referencia/Búsqueda**
- Abrir registros específicos
- Información de referencia cruzada
- Cerrar cuando termine, reutilizar para siguiente búsqueda

**Pestaña 4+: Tareas en Segundo Plano**
- Exportaciones largas generándose
- Creación de PDF en progreso
- No cerrar hasta completar

### Operadores de Búsqueda

**Técnicas de búsqueda avanzadas:**

**Coincidencia Exacta:**
```
Búsqueda: "Maria Garcia"
(Con comillas: Solo frase exacta)
```

**Comodín:**
```
Búsqueda: Gar*
(Encuentra: Garcia, Garza, Garciela, etc.)
```

**Múltiples Términos:**
```
Búsqueda: Garcia Centro Beca
(Encuentra registros que coinciden con los tres términos)
```

**Excluir Términos:**
```
Búsqueda: Garcia -Centro
(Encuentra Garcia pero no en Centro)
```

### Expansión de Texto

**Acelere la escritura repetitiva:**

Cree atajos que se expandan a texto completo:

**Ejemplos:**
```
;addr → Calle Juárez #123, Col. Centro, C.P. 48300
;beca → Programa Federal de Becas para el Bienestar Benito Juárez
;recibido → Su reporte ha sido recibido y está siendo procesado.
;firma → Atentamente,\n[Su Nombre]\n[Su Título]\nDepartamento de Programas Sociales
```

**Herramientas:**
- Mac: Reemplazo de Texto (Preferencias del Sistema)
- Windows: AutoHotkey, PhraseExpress
- Chrome/Firefox: Extensiones de expansión de texto

## Consejos para Ahorrar Tiempo

Pequeñas mejoras que se suman a grandes ahorros de tiempo.

### Optimización de Entrada de Datos

**Antes de Comenzar:**
- Pre-organizar todos los documentos
- Verificar que toda la información sea legible
- Verificar datos faltantes
- Tener materiales de referencia listos

**Durante la Entrada:**
- Usar Tab, no mouse, para navegar campos
- Escribir en menús desplegables (no desplazarse)
- Copiar-pegar números CURP/folio (evitar errores)
- Validar mientras avanza (no esperar error de envío de formulario)

### Árboles de Decisión para Escenarios Comunes

**Cuando un Ciudadano Llama Sobre Estado del Programa:**
```
1. Obtener número de folio → Buscar en Programas
2. ¿Sin folio? → Obtener CURP → Buscar en Ciudadanos → Ver programas
3. ¿Encontrado?
   Sí → Dar actualización de estado
   No → "No está en sistema, vamos a crear solicitud"
```

**Al Procesar Solicitud de Programa:**
```
1. ¿Todos los docs cargados?
   No → Cambiar estado a Pendiente + Nota: "Falta [X]" + Notificar ciudadano
   Sí → Continuar

2. ¿Cumple elegibilidad?
   No → Cambiar estado a Negada + Anotar razón + Notificar
   Sí → Continuar

3. ¿Presupuesto disponible?
   No → Poner en lista de espera
   Sí → Aprobar + Asignar folio + Notificar
```

## Errores Comunes a Evitar

Aprenda de los errores de otros.

### Errores de Entrada de Datos

**Entrada de CURP:**
- ❌ No escribir de memoria (alta tasa de error)
- ✓ Copiar de documento oficial
- ✓ Validar formato antes de enviar

**Selección de Calle/Colonia:**
- ❌ No crear duplicado si existe similar
- ✓ Buscar exhaustivamente primero
- ✓ Verificar variaciones de capitalización

**Cambios de Estado:**
- ❌ No cambiar sin notas
- ✓ Siempre documentar por qué
- ✓ Incluir próximos pasos

### Errores de Flujo de Trabajo

**Búsqueda:**
- ❌ No desplazarse por miles de registros
- ✓ Usar búsqueda y filtros
- ✓ Ser específico con términos de búsqueda

**Multitarea:**
- ❌ No cambiar entre 10 tareas a medio hacer
- ✓ Terminar una tarea antes de comenzar la siguiente
- ✓ Agrupar tareas similares juntas

**Documentación:**
- ❌ No confiar en la memoria ("Lo documentaré después")
- ✓ Agregar notas inmediatamente
- ✓ Ser específico y detallado

## Guía de Solución de Problemas

Soluciones a problemas comunes.

### Problemas de Rendimiento

**Carga Lenta:**

**Posibles Causas:**
- Conjunto de datos grande (más de 10,000 registros)
- Conexión lenta a Internet
- Caché del navegador lleno
- Demasiadas pestañas abiertas

**Soluciones:**
```
1. Verificar velocidad de Internet (ejecutar prueba de velocidad)
2. Cerrar pestañas innecesarias del navegador
3. Limpiar caché del navegador:
   - Chrome: Ctrl+Shift+Delete
   - Seleccionar "Imágenes y archivos en caché"
   - Limpiar
4. Aplicar filtros para reducir conjunto de datos
5. Reducir filas por página
6. Probar navegador diferente
7. Reiniciar computadora si está muy lento
```

### Problemas de Datos

**No Puede Encontrar Ciudadano:**

**Lista de Verificación:**
```
1. Verificar ortografía del nombre
2. Intentar buscar por CURP en su lugar
3. Intentar búsqueda parcial (solo apellido)
4. Limpiar todos los filtros
5. Verificar que esté en sección correcta (Ciudadanos)
6. Verificar que el ciudadano realmente exista en el sistema
```

**La Dirección No Se Guarda:**

**Causas Comunes:**
```
1. Falta campo requerido
2. Calle no está en sistema → Agregar calle primero
3. Colonia no está en sistema → Agregar colonia primero
4. Formato de código postal incorrecto → Usar 5 dígitos
5. Error de validación → Leer mensaje de error cuidadosamente
```

### Problemas de Búsqueda

**La Búsqueda No Devuelve Nada:**
```
1. Verificar ortografía
2. Intentar menos letras (búsqueda más amplia)
3. Eliminar filtros que podrían excluir resultados
4. Limpiar búsqueda y comenzar de nuevo
5. Verificar que los datos realmente existan
```

## Resumen de Mejores Prácticas

### Hábitos Diarios

**Inicio del Día:**
1. Verificar Dashboard para prioridades
2. Revisar elementos urgentes
3. Planear trabajo del día
4. Limpiar cabos sueltos de ayer

**Durante el Trabajo:**
1. Documentar mientras avanza
2. Guardar frecuentemente
3. Actualizar estados en tiempo real
4. Comunicarse prontamente

**Fin del Día:**
1. Actualizar todos los elementos en proceso
2. Cerrar sesión apropiadamente
3. Limpiar carpeta de descargas
4. Planear el día de mañana

### Estándares de Calidad

**Entrada de Datos:**
- Verificar antes de enviar
- Usar fuentes oficiales
- Copiar-pegar cuando sea posible (reduce errores)
- Verificar dos veces formato de CURP

**Toma de Decisiones:**
- Revisar toda la información relevante
- Documentar razonamiento
- Seguir procedimientos establecidos
- Escalar cuando no esté seguro

**Comunicación:**
- Tono profesional siempre
- Claro y conciso
- Respuestas oportunas
- Documentar todas las interacciones

**Seguridad:**
- Cerrar sesión cuando termine
- Bloquear pantalla cuando se aleje
- Proteger exportaciones
- Reportar preocupaciones de seguridad
