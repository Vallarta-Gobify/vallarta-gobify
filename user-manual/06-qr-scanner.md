# QR Scanner

The QR Scanner feature provides quick, mobile-friendly access to citizen and program information by scanning QR codes with your device's camera.

## What is the QR Scanner?

The QR Scanner allows authorized government staff to:
- Quickly access citizen profiles
- View program enrollment details
- Verify beneficiary status
- Retrieve information in the field
- Confirm document authenticity

## Use Cases

### Field Verification
Government staff conducting field visits can scan QR codes on:
- Beneficiary identification cards
- Program enrollment certificates
- Printed appointment confirmations
- Official documents

### Office Operations
During in-person citizen services:
- Scan ID cards for instant profile lookup
- Verify program enrollment status
- Confirm appointment details
- Quick access to citizen history

### Document Verification
Validate authenticity of:
- Program participation certificates
- Benefit award letters
- Official government documents
- Digital or printed QR codes

## Using the QR Scanner

### First-Time Setup

When accessing the scanner for the first time:

[Screenshot: Camera permission dialog]

**Camera Permission Prompt:**
```
"Gobify would like to access your camera"
[Block] [Allow]
```

**Action Required:**
1. Read the permission request
2. Click/Tap "Allow"
3. Permission saves for future sessions

**If You Accidentally Blocked:**
1. Go to browser settings
2. Find site permissions for Gobify
3. Enable camera access
4. Refresh the QR Scanner page

### Scanning a QR Code

[Screenshot: QR Scanner interface with camera viewfinder and targeting guide]

**Scanner Interface:**

The interface shows:
- **Camera viewfinder**: Live camera feed
- **Targeting guide**: Square or crosshair showing scan area
- **Instructions**: "Align QR code within frame"
- **Status**: "Ready to scan" or "Scanning..."

**Scanning Process:**

```
1. Open QR Scanner
2. Hold device steady
3. Point camera at QR code
4. Align QR code within the targeting guide
5. Wait for automatic detection (1-2 seconds)
6. Successful scan: Automatic redirect
7. Information loads immediately
```

**Scanning Tips:**

**Distance:**
- Hold device 6-12 inches (15-30 cm) from QR code
- Too close: Code appears blurry
- Too far: Code too small to scan
- Adjust distance until code is clear in viewfinder

**Lighting:**
- Ensure adequate lighting
- Avoid shadows on QR code
- Avoid glare from reflective surfaces
- Use device flashlight if needed

**Steadiness:**
- Hold device still
- Brace against table or wall if shaky
- Take a breath and stabilize
- Auto-scan activates when code is clear

**Alignment:**
- Center QR code in targeting guide
- Entire code must be visible
- Keep code level (not tilted)
- Maintain focus until scan completes

## Troubleshooting

### Camera Not Working

[Screenshot: Error message "Cannot access camera"]

**Possible Causes:**
1. Camera permission denied
2. Camera in use by another app
3. No camera available
4. Browser doesn't support camera access

**Solutions:**

**Check Permissions:**
```
1. Click lock icon in address bar
2. Find Camera permission
3. Ensure set to "Allow"
4. Refresh the page
5. Try scanning again
```

**Close Other Apps:**
```
1. Close other apps using camera (Zoom, etc.)
2. Return to Gobify
3. Refresh QR Scanner page
4. Camera should now work
```

**Try Different Browser:**
```
1. If Safari not working, try Chrome
2. Install updated browser if needed
3. Open Gobify in new browser
4. Access QR Scanner
```

**Restart Device:**
```
1. Close all apps/browser tabs
2. Restart your computer/phone
3. Reopen Gobify
4. Try scanner again
```

### QR Code Won't Scan

**Possible Reasons:**
1. QR code is damaged or unclear
2. Poor lighting conditions
3. Code too small or too far away
4. Camera focus issues

**Solutions:**

**Improve Lighting:**
```
1. Move to better-lit area
2. Turn on device flashlight (mobile)
3. Avoid shadows on code
4. Reduce glare from overhead lights
```

**Adjust Distance:**
```
1. Move closer to code
2. Or back away if too close
3. Aim for 6-12 inches away
4. Ensure entire code visible in frame
```

**Clean Camera Lens:**
```
1. Wipe camera lens with soft cloth
2. Remove fingerprints and smudges
3. Try scanning again
```

**Try Different Angle:**
```
1. Tilt device slightly
2. Avoid direct reflections
3. Scan from straight-on position
4. Rotate code if necessary
```

**Manual Entry Alternative:**
```
1. If code won't scan, note any visible text/numbers
2. Exit scanner
3. Search citizen/program manually
4. Use CURP or folio number
```

### Scans But Loads Wrong Information

**Possible Issues:**
1. QR code links to outdated data
2. Citizen/program was deleted
3. System synchronization delay

**Solutions:**

**Verify QR Code Source:**
```
1. Check when document was issued
2. Ensure it's official Gobify QR code
3. Confirm citizen/program still active
```

**Manual Search:**
```
1. Note citizen name from physical document
2. Search manually in Ciudadanos section
3. Verify information matches
4. Report discrepancy if found
```

**Report Issue:**
```
1. Note QR code details
2. Take photo of code and result
3. Contact IT support or supervisor
4. May need QR code regeneration
```

### Scanner Page Won't Load

**Solutions:**

**Check Internet Connection:**
```
1. Verify WiFi or data connection active
2. Try loading another page
3. Reconnect to network if needed
4. Refresh QR Scanner page
```

**Clear Browser Cache:**
```
1. Go to browser settings
2. Clear browsing data/cache
3. Close and reopen browser
4. Log back into Gobify
5. Try scanner again
```

**Update Browser:**
```
1. Check for browser updates
2. Install latest version
3. Restart browser
4. Access scanner
```

## Security and Privacy

### Secure Scanning

**Best Practices:**
- Only scan official government-issued QR codes
- Verify source of document before scanning
- Don't scan unknown or suspicious codes
- Report unofficial or modified codes

**Security Warnings:**

The scanner may display warnings:
```
"Warning: This QR code appears to be modified"
[Cancel] [Scan Anyway]
```

**Action**: Choose Cancel unless you're certain the code is legitimate.

### Data Privacy

**Information Protection:**
- Scanner only works when logged into Gobify
- Requires proper user authentication
- Scanned data subject to role-based permissions
- Access is logged for audit purposes

**Camera Privacy:**
- Camera activates only when scanner page is open
- No images are stored
- No recording occurs
- Close scanner page to deactivate camera

**What Gets Logged:**
- Date and time of scan
- User who performed scan
- Which citizen/program was accessed
- No camera images stored

## Advanced Features

### Batch Scanning

For processing multiple citizens quickly:

```
1. Open QR Scanner
2. Scan first QR code
3. View/verify information
4. Click "Back to Scanner" button
5. Scanner immediately ready for next code
6. Scan next code
7. Repeat for all citizens
```

**Use Cases:**
- Check-in at program event
- Distribution verification
- Field survey data collection
- Appointment validation

### Offline QR Codes

Some QR codes work with limited functionality offline:

**Online (Full Features):**
- Complete profile information
- Real-time status updates
- Ability to edit records
- Access to all linked data

**Offline (Limited):**
- Basic information embedded in code
- Name, ID number
- Program enrollment status
- No editing capability

**Note**: Most features require internet connection. Offline mode is minimal and for emergency use only.

## Common QR Scanner Workflows

### Workflow: Verify Beneficiary at Distribution Event

```
1. Beneficiary arrives at distribution
2. Presents program enrollment card
3. Open QR Scanner on tablet
4. Scan QR code on card
5. Profile loads showing:
   - Name matches person
   - Program enrollment is active
   - Eligibility confirmed
6. Provide benefit/service
7. Mark as distributed in system
8. Scan next beneficiary
```

### Workflow: Field Visit Verification

```
1. Arrive at citizen's home for field visit
2. Ask citizen for ID or enrollment card
3. Open QR Scanner on phone
4. Scan code
5. Verify address matches
6. Review program details
7. Update notes with visit results
8. Complete field visit form
```

### Workflow: Office Check-In

```
1. Citizen arrives for appointment
2. Presents confirmation letter with QR code
3. Scan code at reception desk
4. Confirm appointment details
5. Alert assigned staff member
6. Direct citizen to waiting area
7. Log check-in time
```

### Workflow: Document Authentication

```
1. Citizen submits certificate for verification
2. Document has QR code
3. Scan code
4. System shows:
   - Document is authentic
   - Issue date
   - Issuing office
   - Valid status
5. Accept document as valid
6. Proceed with transaction
```

## Best Practices

### Preparation

**Before Using Scanner:**
1. Ensure device is charged
2. Test camera is working
3. Check internet connection
4. Clean camera lens
5. Find well-lit area

### Scanning Technique

**For Best Results:**
1. Use rear camera on mobile devices (higher quality)
2. Hold device steadily
3. Align code within guide
4. Wait for auto-scan (don't rush)
5. Give clear visual confirmation before moving on

### Efficiency Tips

**High-Volume Scanning:**
1. Set up dedicated scanning station
2. Use tablet with stand for stability
3. Ensure consistent lighting
4. Prepare citizens: "Please have QR code ready"
5. Use "Back to Scanner" for quick succession
6. Take breaks every 50-100 scans

### Error Handling

**When Things Go Wrong:**
1. Don't panic - use manual search as backup
2. Note which codes fail repeatedly
3. Report technical issues to IT
4. Keep physical documents as fallback
5. Document workarounds used
