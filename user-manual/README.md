Welcome to the official user manual for the Gobify Admin Portal, a comprehensive government administration platform designed to streamline citizen services, social program management, and geographic data organization.

This manual provides step-by-step instructions for government staff to effectively use the Gobify Admin Portal. Whether you're new to the system or need a refresher on specific features, you'll find clear, practical guidance throughout these pages.

## Who Should Use This Manual

- Government administrators
- Social program coordinators
- Citizen services staff
- Geographic data managers
- Any authorized government personnel using the Gobify platform

## User Roles and Permissions

Gobify supports different user access levels:

- **Federal-only users**: Access to federal social programs only
- **Local-only users**: Access to local (municipal) social programs only
- **Full access users**: Complete access to both federal and local programs

Your available features will depend on your assigned role. If you need different permissions, contact your system administrator.
